# Customer Demographics & Loyalty Dashboard

## Project Overview
Developed a Power BI dashboard to analyze customer demographics, purchasing behavior, product preferences, and loyalty segments.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Visualization

## Key Insights
- Female and Male customers contribute nearly equal order volume.
- Electronics and Clothing categories generate the highest orders.
- Average customer age is 36.5 years.
- Order distribution varies significantly across countries.

## Recommendations
- Strengthen loyalty programs for high-value customers.
- Increase focus on high-performing product categories.
- Use demographic insights for targeted marketing campaigns.
